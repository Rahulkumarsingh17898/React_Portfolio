# Rahul Kumar Singh - Developer Portfolio

A modern, responsive personal portfolio website showcasing 5+ years of enterprise software engineering experience, complex system designs, and full-stack personal projects.

## 🛠 Tech Stack

* **Framework:** React (scaffolded with Vite)
* **Language:** TypeScript
* **UI Library:** Material UI (MUI)
* **Animations:** Framer Motion
* **Deployment:** GitHub Pages / Vercel (Planned)

## 🚀 Features

* **Responsive Design:** Fully fluid layouts optimized for mobile, tablet, and desktop viewing.
* **Modern Animations:** Smooth page transitions and scroll-based reveal effects powered by Framer Motion.
* **Component-Driven:** Clean architecture using reusable React components and strict TypeScript interfaces.
* **Featured Work Section:** Detailed breakdowns of enterprise-scale data migrations, architecture (HLD/LLD), and independent full-stack builds like TaskTracker.

## 💻 Local Development

To run this project locally, ensure you have Node.js installed, then execute the following commands:

1. Clone the repository:
git clone https://github.com/Rahulkumarsingh17898/portfolio-website.git

2. Navigate into the directory:
cd portfolio-website

3. Install the dependencies:
npm install

4. Start the development server:
npm run dev
